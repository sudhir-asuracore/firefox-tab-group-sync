# Security Policy

## Supported Versions

Only the latest released version is actively maintained for security updates.

## Reporting a Vulnerability

Please report security issues privately to:

- Email: support@sudhirnakka.com
- Or open a [GitHub Security Advisory](https://github.com/sudhir-asuracore/firefox-tab-group-sync/security/advisories/new)

We will acknowledge receipt within 72 hours and strive to provide a fix or mitigation as soon as possible. Please do not disclose the issue publicly until a fix is released.
